# 第二章 走进科学实验室

### 实验器材及其用途

- 学校实验室的设备、仪器和试剂分类摆放于实验室的柜子内，存放点必须符合环境要求；一些特殊且具有危险性的试剂或器材的摆放位置具有特殊的需求

#### 实验仪器注意事项

- 可以直接加热的仪器：试管、坩埚、蒸发皿
- **烧杯**不可直接加热，加热时应放置在石棉网上，使其均匀受热
- 试管
  -    加热试管时，要用**试管夹**夹取，夹在离试管1/3 处，管口不能对着人，要使其受热均匀，外壁要干燥，防止骤冷使试管炸裂
  -    加热**液体**时，试管内**液体体积不超过试管容积的1/3**，**管口向上倾斜**，与桌面呈45°角
  -    加热**固体**时，**管口略向下倾斜**
  -    放入**块状固体**时，应该让试管**平放**
- 滴管
  - 胶头在上，管口在下
  - 滴管口不能伸入受滴容器
  - 滴灌用后应立即冲洗，严禁使用未经洗涤的滴管吸取其他试剂

- 量筒
  - 刻度由下而上，无“0”刻度
  - 不可用作反应容器，**不可加热**，也**不能用于稀释或配制溶液**
  - 只可以用于测量一种溶液
  - 读数时量筒必须平放，实现要与两桶内液体凹液面的最低处保持水平
  - 需根据所量取的液体体积选择合适规格的量筒，减少误差
- 放大镜
  - 通过调整放大镜和眼睛的距离和调整放大镜和物体的距离可获得大又清晰的图像
- 闻试剂的味道时，鼻子靠近化学室物品，用手在鼻尖煽动。只有稍微闻到气味即可。千万不能直接闻
- 摆放药品时，标签应向外
- 倾倒试液时，药品外的标签必须向着手心
- **安全卫生防护用具**：实验衣、护目镜、过滤或防护面罩、防护手套、洗眼器、冲淋设备等

| 用具          | 用途                                       |
| ------------- | ------------------------------------------ |
| 试管架        | 用于盛放试管                               |
| 试管          | 用作少量试剂的反应容器，在常温或加热时使用 |
| 试管夹        | 用于夹持试管                               |

### 实验室中的废弃物处理

- 做实验时要正确处理实验过程中产生的废液、固体废物和废气，不能任意排放

| 用具           | 用途                                                         |
| -------------- | ------------------------------------------------------------ |
| 通风橱、抽气柜 | 减少实验者和**有害气体**的接触                               |
| 收集槽         | 用收集实验室过程产生的**废液废水**等，具备良好的抗腐蚀性能，可盛放酸、碱、盐、有机溶剂等废液 |

### 本生灯

![本生灯的结构](https://i.ibb.co/CQrqHNt/image.png)

1. 关掉气孔
2. 将点燃的火柴移近灯管口
3. 旋开煤气
4. 慢慢旋开气孔
5. 使用完毕后，先把气孔关上，然后关掉煤气掣

### 火焰类型

|                |            有光火焰            |        无光火焰        |
| -------------- | :----------------------------: | :--------------------: |
| 例子           | 蜡烛焰、本生灯气孔关闭时的火焰 | 本生灯气孔打开时的火焰 |
| 火焰颜色       |              黄色              |          蓝色          |
| 温度           |              较低              |          较高          |
| 油烟（黑烟）   |               多               |           少           |
| 用来做加热实验 |             不适合             |          适合          |
| 燃气           |      不完全燃烧，浪费能源      |   完全燃烧，节省能源   |

- 当厨房用的煤气炉的火焰橙黄色时，需将气孔开大，让更多的空气进入，使煤气充分燃烧，火焰的颜色也会转变成蓝色，温度比较高，适合使用
- 外焰：无光、温度最高的部分
  - 加热物体时应该选择外焰部位
- 内焰：黄色、最亮
- 焰心：未燃烧的气体、最暗

### 实验室的安全

#### 实验室安全守则

| 遵守                                       | 避免                         |
| ------------------------------------------ | ---------------------------- |
| 遵守老师的指导并进行试验                   | 不能擅自进入实验室           |
| 保持实验室的整洁                           | 不得在实验室中饮食           |
| 实验后，把实验器具洗干净，并放回指定的地方 | 不能用舌头尝试实验室中的药品 |
|                                            | 不得在实验室中嬉闹           |

### 意外的紧急应变措施

- 报告老师 & 保持镇定

| 意外事故           | 正确的处理方法             |
| ------------------ | -------------------------- |
| 烧伤或烫伤         | 用大量冷水冲洗受伤处       |
| 被化学试剂灼伤     | 用缓缓流水冲洗1分钟以上    |
| 被化学试剂溅入眼睛 | 用洗眼液冲洗数分钟         |
| 轻微割伤           | 清理伤口后用创可贴贴在伤口 |
| 打破了水银温度计   | 撒上硫磺粉后再扫起来处理   |
| 伤害比较严重       | 到诊所或医院就医           |

#### 失火原因及灭火方法

| 失火原因             | 灭火方法                             |
| -------------------- | ------------------------------------ |
| 容器内的易燃物着火   | 用玻璃纤维布或灭火毯盖灭             |
| 人衣服着火           | 用水龙头浇水灭火                     |
| 导线、电器或仪器着火 | 先切断电源，然后选择相应的灭火器灭火 |

### 操作灭火器的步骤

1. 右手提灭火器
2. 拔下保险销(xiāo)
3. 一手握喷管，另一只手捏紧压把，将灭火器喷嘴向着燃烧区
4. 距离火焰2m的地方用力压压把，喷管对着火焰根部左右摆动
